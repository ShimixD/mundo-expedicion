# mundo-expedicion
## Necesario:
Crea tu archivo .env y coloca estos datos
```js
DISCORD_CLIENTID=
DISCORD_TOKEN=
DISCORD_CLIENTSECRET=
DISCORD_CALLBACKURL=
MONGODB=
APIPASS=
PORT=
```
## Comenzar a encerderlo:
Lo que tienes que hacer para encenderlo es colocando estos comandos en la consola:
```js
npm i
npm run dev // o npm start
```
